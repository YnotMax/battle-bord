# Sobre o Projeto (About)

O **I M O R T A I S Battle Board** é mais do que um simples visualizador de killboards; é uma ferramenta de **inteligência competitiva** desenhada para dominar o cenário ZvZ do Albion Online.

### 🎯 O Propósito
No Albion Online, os dados de batalha são densos e muitas vezes difíceis de interpretar em tempo real. Os Oficiais e Callers precisam de respostas rápidas:
- *Quem são nossos jogadores mais consistentes?*
- *Nossa composição está equilibrada para o meta atual?*
- *Por que perdemos aquele último engage?*

Este hub responde a essas perguntas transformando JSONs brutos da API em **estratégia visual**.

### 🧬 DNA Tecnológico
O projeto foi construído sob três pilares:
1.  **Velocidade**: Utilizando Next.js com Turbopack para garantir que o painel carregue instantaneamente, mesmo com milhares de registros.
2.  **Precisão**: Um motor de processamento em Python que "garimpa" cada morte (kill event) para somar danos e curas reais, superando as limitações das ferramentas genéricas.
3.  **Estética Imersiva**: Uma interface que respeita a cultura gamer, utilizando uma linguagem visual futurista que faz o usuário se sentir dentro de um centro de comando tático.

### 🛠️ Core Operacional
*   **Data Mining**: O crawler ignora o ruído e foca apenas nos dados atômicos dos participantes da guilda.
*   **Heuristic Role Detection**: Um sistema que analisa o equipamento (Main Hand e Armor) para classificar automaticamente a função de cada jogador no campo de batalha.
*   **Tactical Mentorship**: Um algoritmo que compara a performance individual com a média da guilda, gerando insights automáticos para melhoria de gear e posicionamento.
